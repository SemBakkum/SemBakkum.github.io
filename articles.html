Taskmanagers

Gulp

Taskmanagers are occuring more and more, but what are these taskmanagers or task runners and why should we use them? Most of you have probably heard of Grunt or Gulp and maybe a lot more like Webpack, Broccoli or Cake. If you find yourself as a developer doing time consuming and boring tasks then a task runner could be the solution for you. For example task runners can do the following tasks for you:

  • Minification and concatenation of Javascript and CSS files
  • CSS preprocessing
  • Image optimisation
  • Unit testing and linting
In this article I will be focussing on one of these task runners named Gulp and guess what, setting up a task runner is quite simpel.
